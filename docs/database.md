# Database Tables

Recommended PostgreSQL/Supabase tables:

- `users`
- `projects`
- `services`
- `service_categories`
- `gallery`
- `gallery_categories`
- `blogs`
- `blog_categories`
- `testimonials`
- `career_jobs`
- `career_applications`
- `contact_leads`
- `calculator_reports`
- `downloads`
- `settings`
- `seo_metadata`

Add row-level security policies for public reads on published content and authenticated admin writes.
