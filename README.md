# Siddhi Electricals

Premium full-stack web platform for Siddhi Electricals, an electrical contracting and consultancy company.

## Structure

- `frontend/` - Next.js App Router website, calculators, admin dashboard shell, SEO metadata, Supabase client helpers.
- `backend/` - API/service layer skeleton for future Express or serverless deployment.
- `docs/` - Architecture, database, and deployment notes.

## Quick Start

```bash
cd frontend
npm install
npm run dev
```

Copy `frontend/.env.example` to `frontend/.env.local` and add Supabase values when backend features are connected.
